# Визуализация данных для получения информации о тенденциях в здравоохранении

Данный проект посвящён анализу и визуализации данных в области здравоохранения. Целью работы является разработка программы, позволяющей выявить и проанализировать ключевые тенденции в здравоохранении с использованием визуализации данных.

## Содержание проекта

Проект состоит из следующих файлов:

- **healthcare_dataset.csv**: Набор данных, содержащий информацию о госпитализациях, выписках и других аспектах здравоохранения.

- **Курсовая_работа__Визуализация_данных_для_получения_информации_о_тенденциях_в_здравоохранении.html**: HTML-версия, которая включает все разделы и графики.

- **Курсовая_работа__Визуализация_данных_для_получения_информации_о_тенденциях_в_здравоохранении.ipynb**: Jupyter Notebook, в котором реализован процесс обработки данных, а также сама визуализация.

## Установка библиотек

Для работы с проектом вам понадобятся следующие библиотеки:

- pandas
- numpy
- matplotlib
- seaborn
